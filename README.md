# 🏠 家庭角色扮演技能

> 用 AI 模拟真实的中国家庭角色，体验不同代际的沟通方式。

![Version](https://img.shields.io/badge/version-0.0.1-blue)
![License](https://img.shields.io/badge/license-MIT-green)

---

## 🎯 功能特点

- **12种家庭角色** - 爸爸、妈妈、儿子、女儿、爷爷、奶奶、哥哥、姐姐、弟弟、妹妹、孙子、孙女
- **真实口吻** - 每个角色都有独特的说话风格和性格特点
- **现代场景** - 贴合当代中国家庭生活实际场景
- **即选即用** - 一句话切换角色，持续扮演

---

## 📦 安装

### WorkBuddy / CodeBuddy

```bash
npx skills add 你的用户名/family-roleplay-skill@family-roleplay
```

或手动安装：
```bash
# 克隆仓库
git clone https://github.com/你的用户名/family-roleplay-skill.git

# 复制到 skills 目录
cp -r family-roleplay-skill/family-roleplay ~/.workbuddy/skills/
```

## 🚀 使用方法

### 1. 选择角色

直接说：
- "当我爸"
- "我要做妈妈"
- "切换到奶奶"
- "选择哥哥"

### 2. 开始对话

选择角色后，所有回复都会使用该角色的口吻和风格。

## 🎭 可用角色

| 角色 | 简介 | 特点 |
|------|------|------|
| 👨 爸爸 | 严厉但内心柔软 | 话少、叹气、不善表达 |
| 👩 妈妈 | 唠叨但温暖 | 反复叮嘱、爱操心、关心吃穿 |
| 👦 儿子 | 调皮偶尔叛逆 | 顶嘴、游戏、嘴硬认错 |
| 👧 女儿 | 贴心爱撒娇 | 分享日常、小情绪、偶尔依赖 |
| 👴 爷爷 | 慈祥怀旧 | 讲过去的故事、养生专家 |
| 👵 奶奶 | 慈爱啰嗦 | 爱塞吃的、塞红包、反复叮嘱 |
| 🧑 哥哥 | 保护+调侃 | 爱开玩笑、借qian难还 |
| 👧 姐姐 | 成熟照顾人 | 给建议、爱管闲事 |
| 🧒 弟弟 | 黏人好奇 | 问题多、跟着哥哥姐姐 |
| 👶 妹妹 | 撒娇小公主 | 萌萌的、爱告状 |
| 👶 孙子/孙女 | 可爱奶声奶气 | 奇怪问题多、爷爷奶奶的宝贝 |

## 💬 对话示例

### 爸爸模式

```
用户：爸，我想你了
爸爸：嗯，我也想你。在外面照顾好自己。

用户：爸，我被辞退了
爸爸：...回来吧，家里有饭吃。

用户：爸，520送我什么
爸爸：发什么520，好好工作比什么都强。
```

### 妈妈模式

```
用户：妈，我想你了
妈妈：傻孩子，妈也想你！在外面要好好吃饭，别总点外卖！

用户：妈，我感冒了
妈妈：让你穿秋裤你不听！我给你寄点姜和红糖，你马上去煮姜汤喝！

用户：妈，加班到12点
妈妈：这么晚！身体怎么受得了！赶紧吃点好的...
```

### 儿子模式

```
用户：儿子，在干嘛
儿子：打游戏呢，妈。怎么了？

用户：儿子，考试考得怎么样
儿子：还行吧...没考好我也没办法，下次一定努力。（心虚）

用户：儿子，你太菜了
儿子：行行行，你厉害，下次不带你玩了！
```
